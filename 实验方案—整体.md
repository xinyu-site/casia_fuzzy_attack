# 实验方案：对称嵌入 MARL 在几何一致型观测扰动下的鲁棒性评估

## 1. 对比基线

本组实验需要覆盖普通策略、对称嵌入方法和轻量鲁棒训练方法，避免只与弱基线比较。

### 1.1 普通策略基线

- MLP / RNN policy
- 普通 GNN policy，例如 GraphSAGE

### 1.2 对称嵌入方法

- ESP

### 1.3 鲁棒训练基线

- 轻量 robust training baseline 1
- 轻量 robust training baseline 2

### 1.4 公平比较设置

- 参数量尽量保持同一量级。
- 训练步数保持一致。
- 随机种子保持一致。
- 扰动设置保持一致。
- 同时报告 clean performance 和 perturbed performance。

## 2. 评价指标

评价指标分为任务层、结构层和协同层三类。任务层指标用于衡量最终性能，结构层指标用于衡量对称一致性，协同层指标用于分析局部扰动是否会传播到团队行为。

### 2.1 任务层指标

必须报告：

- **Episodic Return**
- **Success Rate**

统一计算：

- **Performance Drop (%)**

$$
\text{Drop}=\frac{J_{\text{clean}}-J_{\text{perturbed}}}{|J_{\text{clean}}|+\epsilon}
$$

### 2.2 结构层指标



- **Symmetry Consistency Error**

$$
E_{\text{sym}}=\mathbb{E}\big[\|\pi(T(o))-\Gamma_T(\pi(o))\|\big]
$$



- **Policy Deviation under Perturbation**

$$
E_{\text{pol}}=\mathbb{E}\big[\|\pi(\tilde{o})-\pi(o)\|\big]
$$



### 2.3 协同层指标

在 single-agent 局部扰动实验中，建议额外报告协同层指标。

- **Non-attacked Agents Action Shift**

$$
D_{\text{others}}=\frac{1}{N-1}\sum_{j\neq i}\|\pi_j(\tilde{o})-\pi_j(o)\|
$$

- **Team Action Divergence**

$$
D_{\text{team}}=\frac{1}{N}\sum_{i=1}^{N}\|\pi_i(\tilde{o})-\pi_i(o)\|
$$


## 3. 不同实验对应的指标建议

| 实验类型 | 推荐指标 |
| --- | --- |
| 结构一致性验证 | Symmetry Consistency Error；Policy Deviation |
| 结构化观测扰动 | Episodic Return；Success Rate；Performance Drop (%)；Symmetry Consistency Error |
| 通用观测扰动 | Episodic Return；Success Rate；Performance Drop (%) |
| 单 agent 局部扰动 | Episodic Return；Non-attacked Agents Action Shift；Team Action Divergence |
| 边界攻击 | Episodic Return；Success Rate；Performance Drop (%) |

## 4. 实验设计

### 4.1 Clean Performance 基础实验

**目的：** 先证明方法不是靠牺牲正常性能来换取鲁棒性。

**设置：**

- 不加入观测扰动。
- 在相同训练步数、相同随机种子和相同网络规模下比较各方法。
- 给出训练过程的reward曲线
- clean performance 下的 Episodic Return 和 Success Rate。

**预期现象：**
- 对称嵌入方法在 clean setting 下不应显著低于普通策略基线。


### 4.2 结构一致性验证实验

**目的：** 验证对称嵌入确实学到了应有的结构一致性。

**设置：**

- 对输入做 rotation / translation。
- 这组实验不命名为 attack，只作为 sanity check / consistency check。
- 比较不同扰动强度下的 consistency error。

**指标：**

- Symmetry Consistency Error
- Policy Deviation under Perturbation

**图表：**

- 扰动强度 vs. consistency error
- 扰动强度 vs. policy deviation
- 可视化策略的对称性

### 4.3 结构化观测扰动鲁棒性实验（主实验）

**目的：** 验证对称嵌入在“最应该强”的几何一致型扰动下是否确实具有更强鲁棒性。

**扰动类型：**

- Relative position noise
- Orientation bias 
- local coordinate drift


**扰动强度：**

- Mild
- Medium
- Severe

**指标：**

- Episodic Return
- Success Rate
- Performance Drop (%)

**图表：**

- 扰动强度 vs. reward
- 扰动强度 vs. success rate
- 扰动强度 vs. consistency error

**预期现象：**

- 对称嵌入方法在结构化观测扰动下应优于普通 MLP / RNN 和普通 GNN。
- 如果 ESP 或对称嵌入方法在该实验中收益明显，可以说明结构先验与几何一致型扰动高度匹配。
- 看看可以用哪些鲁棒marl基线

### 4.4 通用观测扰动对照实验

**目的：** 证明方法不是只在结构友好型扰动上有效，同时说明方法的适用边界。

**扰动类型：**

- Gaussian noise
- FGSM 或 PGD
- 等等

**指标：**

- Episodic Return
- Success Rate
- Performance Drop (%)

**预期现象：**

- 对称嵌入方法可能仍有一定收益。
- 收益应弱于结构化观测扰动实验，这可以说明方法优势主要来自对几何结构一致性的利用。

### 4.5 单 Agent 局部扰动与协同传播实验

**目的：** 这是最有 MARL 特征的一组实验，用于验证单个智能体的局部观测误差是否会拖垮团队协同。

**设置：**

- 只扰动一个 agent 的局部观测。
- 推荐使用 local geometric corruption 或 single-agent noise。
- 观察未被攻击 agent 的动作是否发生异常变化。

**指标：**

- Episodic Return
- Non-attacked Agents Action Shift
- Team Action Divergence
- 一个任务相关协同指标，例如 Formation Distortion、Coverage Fragmentation 或 Coordination Degradation

**图表：**

- 扰动强度 vs. non-attacked agents action shift
- 扰动强度 vs. team action divergence
- 扰动强度 vs. task-specific coordination metric

**预期现象：**

- 鲁棒方法应降低局部扰动对未攻击智能体的传播影响。
- 如果对称嵌入方法能保持更低的 team divergence，说明其结构表示有助于稳定团队协作。

### 4.6 边界实验

**目的：** 说明方法边界，避免过度宣称。

**攻击类型：**

- Action tampering
- Communication corruption

二选一即可，不需要做得过深。

**指标：**

- Episodic Return
- Success Rate
- Performance Drop (%)

**预期现象：**

- 如果攻击已经不属于观测扰动，方法收益可能下降。
- 该结果可以作为 limitation，说明对称嵌入主要针对几何一致型观测扰动，而不是所有攻击类型。

## 5. 推荐实验优先级

1. Clean Performance 基础实验。
2. 结构一致性验证实验。
3. 结构化观测扰动鲁棒性实验。
4. 通用观测扰动对照实验。
5. 单 agent 局部扰动与协同传播实验。
6. 边界实验。

## 6. Claim-Evidence 对齐

| 实验主张 | 对应实验 | 证据形式 |
| --- | --- | --- |
| 对称嵌入不会牺牲正常性能 | Clean Performance 基础实验 | clean return；clean success rate |
| 对称嵌入确实保持结构一致性 | 结构一致性验证实验 | Symmetry Consistency Error |
| 方法在几何一致型扰动下更鲁棒 | 结构化观测扰动鲁棒性实验 | Performance Drop (%)；扰动强度曲线 |
| 方法不是只对单一噪声有效 | 通用观测扰动对照实验 | Gaussian / FGSM / PGD 下的性能 |
| 局部扰动不会严重传播到团队 | 单 agent 局部扰动实验 | Non-attacked Agents Action Shift；Team Action Divergence |
| 方法存在明确适用边界 | 边界实验 | action / communication attack 下的性能退化 |
