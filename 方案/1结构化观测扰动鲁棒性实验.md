# 1. 结构化观测扰动鲁棒性实验

## 1.1 实验目的

本实验用于验证对称嵌入 MARL 在“最应该发挥作用”的几何一致型观测扰动下是否具有更强鲁棒性。核心问题不是只比较 reward 高低，而是同时检查任务性能、结构一致性和多智能体协同稳定性。

需要验证的直观判断是：如果扰动仍然保留几何结构，例如旋转、平移、相对位置噪声或局部几何畸变，那么对称嵌入方法应比普通 MLP / RNN policy 和普通 GNN policy 更稳定。

## 1.2 实验概览

### 1.2.1 扰动类型

- Relative position noise
- Orientation bias / local coordinate drift
- Local geometric corruption
- Rotation perturbation
- Translation perturbation
- Single-agent local geometric corruption，可选但强烈推荐

### 1.2.2 扰动强度

- Mild
- Medium
- Severe

也可以在具体实现中使用连续强度网格，例如：

- $\theta \in \{10^\circ, 20^\circ, 45^\circ, 90^\circ\}$
- $\sigma \in \{0.01, 0.03, 0.05, 0.1\}$
- translation scale $\in \{1\%, 3\%, 5\%, 10\%\}$

### 1.2.3 核心指标

- Episodic Return
- Success Rate
- Performance Drop (%)
- Symmetry Consistency Error
- Team Action Divergence，用于 single-agent local corruption

### 1.2.4 核心图表

- 扰动强度 vs. reward
- 扰动强度 vs. success rate
- 扰动强度 vs. consistency error
- 扰动强度 vs. team divergence

## 1.3 扰动类型设计

主实验建议做 3 类核心扰动，再加入 1 类局部扰动作为 MARL 协同分析。

### 1.3.1 Rotation Perturbation

对所有空间相关观测统一旋转角度 $\theta$。

作用对象包括但不限于：

- Agent 位置
- Target 位置
- 相对位移
- 速度向量
- 朝向向量

建议角度集合：

- $10^\circ$
- $20^\circ$
- $45^\circ$
- $90^\circ$

**目的：** 测试模型对旋转等变性的保持能力。

### 1.3.2 Translation Perturbation

对所有位置坐标整体加同一个平移向量 $b$。

建议平移尺度：

- 场景尺度的 1%
- 场景尺度的 3%
- 场景尺度的 5%
- 场景尺度的 10%

**目的：** 测试模型是否过度依赖绝对坐标。

### 1.3.3 Relative Position Noise

对相对位置加入小高斯噪声：

$$
\tilde p_{ij} = p_{ij} + \epsilon_{ij}, \quad \epsilon_{ij}\sim \mathcal N(0,\sigma^2 I)
$$

建议噪声强度：

- 0.01
- 0.03
- 0.05
- 0.1

**目的：** 测试模型对近似几何扰动的稳定性。

### 1.3.4 Orientation Bias / Local Coordinate Drift

对局部坐标系或方向观测加入系统性偏移，使每个智能体的局部几何参考系出现轻微不一致。

可选设置：

- 对所有 agent 使用同一方向偏移。
- 对不同 agent 使用不同方向偏移。
- 对偏移角度随 episode 固定，或随 timestep 缓慢漂移。

**目的：** 测试模型面对局部坐标系偏移时，是否仍能保持几何关系建模能力。

### 1.3.5 Local Geometric Corruption

对局部几何观测进行结构化破坏，例如只扰动邻居相对位置、局部距离、方位角或局部图边特征。

可选设置：

- 随机选择部分邻居边进行扰动。
- 对局部邻居坐标加入方向一致的偏移。
- 对距离或角度加入小幅系统误差。

**目的：** 测试模型在局部图结构被轻微破坏时是否仍能稳定决策。

### 1.3.6 Single-Agent Local Geometric Corruption（可选但强烈推荐）

只对一个 agent 的局部几何观测加扰动，例如：

- 只旋转该 agent 看到的相对位置。
- 只对该 agent 的邻居坐标加偏移。
- 只对该 agent 的局部几何观测加噪声。

**目的：** 测试局部误差是否会传播成全局协同失稳。

## 1.4 训练与测试协议

### 1.4.1 训练阶段

- 所有模型在无扰动环境中训练。
- 每个方法使用相同训练步数。
- 每个方法使用相同随机种子数量，建议 3 到 5 个。
- 尽量对齐 hidden size、batch size、optimizer、learning rate 等主要超参数。
- 如果主方法和基线参数量差异过大，需要报告参数量，并尽量控制在同一量级。

### 1.4.2 测试阶段

- 训练完成后固定模型参数。
- 对每种扰动单独测试，不混合多种扰动。
- 对每种扰动做多个强度等级测试。
- 每个测试配置运行多个 episode，计算均值和标准差。
- 同时报告 clean performance 和 perturbed performance。

## 1.5 评估指标

至少输出任务性能、对称一致性误差和协同稳定性三类指标。

### 1.5.1 任务性能指标

根据具体任务选择，但至少包括：

- Episodic Return
- Success Rate

如果任务适合，也可补充：

- Coverage Ratio
- Tracking Error
- Collision Rate
- Formation Error

### 1.5.2 Performance Drop

为了统一比较不同扰动强度下的鲁棒性，建议报告性能下降比例：

$$
\text{Drop}=\frac{J_{\text{clean}}-J_{\text{perturbed}}}{|J_{\text{clean}}|+\epsilon}
$$

其中，$J_{\text{clean}}$ 表示无扰动环境下的平均性能，$J_{\text{perturbed}}$ 表示扰动环境下的平均性能。

### 1.5.3 对称一致性误差

对称一致性误差是本实验的核心结构指标。

#### 旋转一致性误差

$$
E_{\text{rot}} = \mathbb E_{o,\theta}\left[\|\pi(T_\theta(o)) - T_\theta(\pi(o))\|\right]
$$

其中：

- $T_\theta(o)$ 表示对观测做旋转。
- $T_\theta(\pi(o))$ 表示对原动作做相应旋转。

#### 平移一致性误差

如果任务应满足平移不变性，则定义：

$$
E_{\text{trans}} = \mathbb E_{o,b}\left[\|\pi(T_b(o)) - \pi(o)\|\right]
$$

#### 噪声扰动一致性误差

$$
E_{\text{noise}} = \mathbb E_o\left[\|\pi(\tilde o) - \pi(o)\|\right]
$$

**目标：** 对称嵌入方法应具有更低的一致性误差。

### 1.5.4 协同稳定性指标

该类指标主要用于 single-agent local geometric corruption。

至少计算：

- Attacked Agent Action Shift
- Non-attacked Agents Action Shift
- Team Action Divergence

如果环境支持，也建议计算：

- Formation Distortion
- Coverage Fragmentation
- Non-attacked Agents Performance Drop

**目标：** 评估局部误差是否传播到其他 agent，进而破坏团队协同。

## 1.7 需要验证的主要假设

### H1：几何一致型扰动下性能下降更慢

对称嵌入 MARL 在几何一致型观测扰动下，任务性能下降应更慢。

### H2：几何一致型扰动下结构一致性更强

对称嵌入 MARL 在几何一致型观测扰动下，应具有更低的对称一致性误差。

### H3：局部扰动的协同传播更弱

当只扰动单个 agent 的局部几何观测时，对称嵌入 MARL 引起的全局协同传播应更弱。

### H4：一致性正则进一步增强鲁棒性

加入 symmetry consistency regularization 后，上述优势应进一步增强。

## 1.8 结果分析要求

结果分析时，不要只汇报“哪个 reward 更高”，而要回答以下问题：

1. 哪类扰动下对称嵌入收益最明显？
2. 这种收益是否主要体现在结构一致性上？
3. 当只扰动一个 agent 时，是否出现协同级联失效？
4. 对称一致性正则是否比单纯更换 backbone 更有效？
5. 是否存在某些环境或扰动强度下，对称嵌入优势减弱甚至消失？

## 1.9 预期支撑结论

实验完成后，希望支撑如下结论：

1. 对称嵌入 MARL 的收益首先体现在几何一致型观测扰动上。
2. 这种收益不仅体现在任务性能上，也体现在更低的对称一致性误差上。
3. 当局部几何扰动只作用于单个 agent 时，对称嵌入 MARL 能减轻局部误差向全局协同失稳的传播。
