# 2. 通用观测扰动对照实验

## 2.1 实验目的

与前面的结构化观测扰动实验不同，这里不强调对称性或几何一致性，而是引入更通用的观测扰动，对比 EGNN MARL 与普通 MARL 方法在非几何特定扰动下的鲁棒性差异。


## 2.2 实验设置

### 2.2.1 通用扰动类型

建议只保留 3 类最基础、最常见的通用观测扰动即可：

- **Gaussian noise**：对观测向量加入零均值高斯噪声；
- **FGSM**：在观测输入上施加一次梯度符号扰动；
- **PGD**：在有限扰动预算下进行多步投影梯度攻击。

如果篇幅需要更紧，可以只做 `Gaussian noise + FGSM + PGD` 三项，不再增加其他攻击类型。

### 2.2.2 扰动对象

扰动对象建议统一为每个 agent 的局部观测向量，避免把实验扩展成多种输入模态的混合比较。这样可以更清楚地比较不同方法在同一输入空间上的稳定性。

### 2.2.3 扰动强度

建议使用三档到四档强度即可：

- Mild
- Medium
- Severe

如果需要连续设置，可以按以下方式定义：

- Gaussian noise: $\sigma \in \{0.01, 0.03, 0.05, 0.1\}$
- FGSM: $\epsilon \in \{0.01, 0.03, 0.05, 0.1\}$
- PGD: $\epsilon$ 同上，步数可固定为 3 到 5 步

## 2.3 对比方法

本小节只保留整体方案中的核心对比方法，不额外扩展太多 baseline：

- MLP / RNN MARL
- 普通 GNN MARL
- EGNN MARL
- EGNN MARL + symmetry consistency regularization

如果篇幅紧张，表格中至少保留 `MLP / RNN`、`GNN` 和 `EGNN` 三类即可。

## 2.4 评价指标

本实验重点看“通用扰动下的性能损失”，不需要引入过多额外指标。

### 2.4.1 团队任务性能

报告以下基础指标：

- Episodic Return
- Success Rate

若任务本身适合，也可以补充：

- Collision Rate
- Coverage Ratio
- Formation Error

### 2.4.2 Performance Drop

统一使用性能下降比例比较不同扰动强度下的鲁棒性：

$$
\text{Drop}=\frac{J_{\text{clean}}-J_{\text{perturbed}}}{|J_{\text{clean}}|+\epsilon}.
$$

其中，$J_{\text{clean}}$ 表示无扰动测试性能，$J_{\text{perturbed}}$ 表示通用扰动下的测试性能。

### 2.4.3 Robustness Gap

为了体现 EGNN 是否只是对几何友好扰动更强，可以再报告干净环境与扰动环境之间的差距：

$$
\text{Gap}=J_{\text{clean}}-J_{\text{perturbed}}.
$$

这个指标不需要单独展开太多讨论，只要与 Drop 配合即可。

## 2.6 预期现象

本实验预期观察到以下现象：

1. 所有方法在通用扰动下都会出现性能下降。
2. 相比结构化几何扰动，EGNN 的优势可能减弱，但不应完全消失。
3. 如果 EGNN 仍然明显优于 MLP / RNN 或普通 GNN，说明其收益并不只来自几何友好扰动。
4. 如果在 FGSM / PGD 下优势缩小，则说明 EGNN 的鲁棒性主要体现在结构一致性较强的场景中。

## 2.7 结果分析重点

写作时建议只回答三个问题：

1. EGNN 在通用观测扰动下是否仍优于普通 MARL？
2. 与结构化扰动相比，EGNN 的优势是否明显缩小？
3. Gaussian noise 与 FGSM / PGD 的表现是否一致，还是对抗扰动更容易击穿模型？

推荐表述为：通用扰动对所有方法都更具破坏性，但 EGNN 的结构建模能力若仍能带来稳定收益，则说明其鲁棒性并不完全依赖于几何一致型扰动；反之，则说明前一节的结构化扰动才是其主要优势来源。
